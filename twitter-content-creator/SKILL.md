# Twitter Content Creator Skill

## Overview
本Skill旨在帮助用户在Twitter（X）中文圈打造个人IP。它基于一套经过实测的"输入 → 输出"SOP，专注于将原始资讯和个人观点快速转化为具备高爆款潜力的内容。

核心价值：
- **速度**：短文 < 20分钟，中线程 < 1.5小时。
- **质量**：强调"核弹级钩子"、情绪共鸣和实用价值。
- **本地化**：专门针对中文X环境的传播规律优化。

## Workflow

### Phase 1: 策略定位 (Strategy)
在开始写作前，协助用户确定内容类型：

| 类型 | 长度/特征 | 目标 | 爆款概率 | 耗时 |
|:---|:---|:---|:---|:---|
| **短文** | 80–220字 (1条) | 占位、引流、制造存在感 | ★★★★★ (最高) | 8–20 min |
| **中线程** | 5–15条 (800–3000字) | 建立专业度、深度教育 | ★★★★☆ | 40–90 min |
| **长文** | 3000+字 (Articles) | 深度信任、品牌建设 | ★★★☆☆ | 2–5 h |

### Phase 2: 内容创作 SOP (Production)
遵循以下5步流程：

1.  **提炼钩子 (30s)**
    *   从资讯+看法中提炼最反常识/扎心的一句话。
    *   *公式*："99%人还在...结果..." / "我用AI干了...结果..." / "别再学...正在害死..."
2.  **列子弹点 (1-2min)**
    *   拆解为3-5个独立点。每个点：痛点/反常识 + 解释 + 情绪/数字。
3.  **AI扩写 (3-10min)**
    *   使用 `assets/prompt_template.md` 中的Prompt进行初稿生成。
4.  **人性化改写 (5-15min)**
    *   *去AI味*：加入口语（"卧槽"、"笑死"）、自嘲、极化观点。
    *   *节奏控制*：短句为主，段落<3行。
5.  **检查优化 (2min)**
    *   钩子是否2秒内吸引人？是否有截图金句？CTA是否自然？

### Phase 3: 结构模板 (Templates)
根据内容类型应用特定结构（详见 `assets/structure_templates.md`）：

*   **短文**：钩子 -> 痛点放大 -> 反转/干货 -> 情绪结尾+CTA
*   **中线程**：钩子+预告 (1) -> 痛点+共鸣 (2-3) -> 核心干货 (4-8) -> 高潮转折 (9-10) -> 总结+CTA
*   **长文**：好奇标题 -> 故事开头 -> 框架拆解 -> 哲学结尾
