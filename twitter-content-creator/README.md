# Twitter Content Creator

一个专注于在 Twitter (X) 中文圈快速打造爆款内容的辅助技能。

## 项目简介

Twitter Content Creator 是基于一套经过实战验证的 SOP（标准作业程序）构建的创作工具。它旨在帮助用户将"资讯 + 个人看法"在极短时间内转化为高质量、高传播率的推文。

核心理念：**输入 → 输出的速度与质量并重**。

## 核心功能

### 🚀 极速创作
- **短文**：8-20分钟产出
- **中线程**：40-90分钟产出
- **长文**：2-5小时产出

### 🎯 爆款策略
- **核弹级钩子**：30秒提炼最扎心/反常识的开头
- **情绪共鸣**：精准把控中文社区的情绪G点
- **结构化模板**：提供短文、线程、长文的专用爆款结构

### 🤖 AI 辅助 + 人性化
- 提供经过调优的 Prompt 模板，让 AI 产出高质量初稿
- 指导进行"去 AI 味"的人性化改写，注入个人风格

## 使用方法

### 1. 准备输入
你需要准备两样东西：
- **资讯**：文章链接、摘要或关键点
- **看法**：你的赞同、反对、吐槽或实验结果

### 2. 选择类型
根据你的目标选择内容类型：
- **短文**（易爆款，适合引流）
- **中线程**（建立信任，深度教育）
- **长文**（品牌建设）

### 3. 执行 SOP
遵循 `SKILL.md` 中的 5 步流程：
1.  提炼钩子
2.  列子弹点
3.  AI 扩写（使用 `assets/prompt_template.md`）
4.  人性化改写
5.  检查优化

## 资源文件

- **SKILL.md**: 完整的操作手册和 SOP 详情
- **assets/prompt_template.md**: 核心 AI 提示词
- **assets/structure_templates.md**: 各类推文的结构公式

## 最佳实践

- **先练短文**：每天尝试用 Prompt 生成 5 条，挑选 2-3 条发布。
- **建立库**：将好的钩子和结构记录下来，形成自己的语感。
- **反直觉**：在中文圈，越是反常识、反直觉的内容，传播力越强。
