# Twitter Content Creator

一个专注于在 Twitter (X) 中文圈快速打造爆款内容的辅助技能。

## 项目简介

Twitter Content Creator 是基于一套经过实战验证的 SOP（标准作业程序）构建的创作工具。它旨在帮助用户将"资讯 + 个人看法"在极短时间内转化为高质量、高传播率的推文。

核心理念：**输入 → 输出的速度与质量并重**。

## 核心功能

### 🚀 极速创作
- **短文**：8-20分钟产出
- **推文串(Thread)**：40-90分钟产出
- **长文**：2-5小时产出

### 🎨 多样化风格
- **通用爆款风格**：犀利、毒舌、情绪化（适合大众传播）
- **技术博主 2.0 风格**：极致干货、高信息密度、结构化、中英混排（适合技术/AI圈）

### 🎯 爆款策略
- **核弹级钩子**：30秒提炼最扎心/反常识的开头
- **情绪共鸣**：精准把控中文社区的情绪G点
- **结构化模板**：提供短文、Thread标准模板、长文的专用爆款结构

### 📅 发布节奏指导
- **分阶段建议**：针对起步期 (0-5k)、增长期 (5k-20k)、成熟期 (20k+) 提供不同的发帖频率建议。
- **黄金时间**：基于中文圈 2026 数据，推荐最佳发布时段。
- **决策辅助**：教你如何根据短文数据决定是否扩写 Thread。

### 🤖 AI 辅助 + 人性化
- 提供经过调优的 Prompt 模板（含Thread专用），让 AI 产出高质量初稿
- 指导进行"去 AI 味"的人性化改写，注入个人风格

## 使用方法

### 1. 准备输入
你需要准备两样东西：
- **资讯**：文章链接、摘要或关键点
- **看法**：你的赞同、反对、吐槽或实验结果

### 2. 选择策略
- **类型**：短文 / Thread / 长文
- **风格**：通用爆款 / 技术博主 2.0

### 3. 执行 SOP
遵循 `SKILL.md` 中的流程：
1.  **创作**：提炼钩子 -> AI 扩写 -> 人性化改写
2.  **发布**：Skill 会根据你的粉丝量级，给出最佳发布时间与频率建议

## 资源文件

- **SKILL.md**: 完整的操作手册和 SOP 详情
- **assets/prompt_template.md**: 核心 AI 提示词（通用 + Thread专用 + 技术博主2.0）
- **assets/structure_templates.md**: 各类推文的结构公式（含Thread标准模板）
- **assets/publishing_guide.md**: 发布节奏与策略指南

## 最佳实践

- **先练短文**：每天尝试用 Prompt 生成 5 条，挑选 2-3 条发布。
- **Thread 进阶**：使用专门的 Thread 模板，按照 7-15 条的节奏进行深度拆解。
- **黄金组合**：每天发短文刷存在感，每周发 4-6 个 Thread 建立深度信任。
- **数据驱动**：昨天的爆款短文，就是今天 Thread 的最佳选题。
