# Twitter Content Structure Templates

## A. 短文 (Short Tweet)
*最高频、最易爆款*

**结构公式**:
1.  **钩子句** (10–20字): 情绪拉满，反常识。
2.  **痛点/现状放大** (1–2句): 制造共鸣。
3.  **反转/干货/金句**: 核心价值交付。
4.  **情绪结尾 + CTA**: 提问/挑衅/引关注。

**示例**:
> AI 绘画 2026 年已经彻底内卷了 (钩子)
> 以前一张图几分钟出片，现在人人都是“ prompt 大神” (现状)
> 结果呢？99% 的人产出都长一个样，审美疲劳到爆炸 (痛点)
> 真正拉开差距的，只有一个东西：你会不会写“反向 prompt” (反转)
> 我昨天试了，把“不要 XXX”写到极致，产出直接起飞 (证明)
> 你还在卷 prompt 数量吗？来评论区说说你最近最拉的图 (CTA)

## B. 推文串 (Thread) 标准模板
*建立信任、深度教育、高转发*

**最佳长度**: 7-15 条
**核心原则**: 每条独立可读（可截图），整体逻辑流畅。

**结构公式**:

| 序号 | 模块 | 内容要点 | 示例/技巧 |
|:---|:---|:---|:---|
| **1/** | **核弹钩子** | 10-20字炸眼球 + 预告价值 | "99% AI 博主还在卷这个，结果它早就没用了" / 反常识 / 痛点 |
| **2-3/** | **痛点放大** | 共鸣拉满 + 数据/故事 | "你是不是也..." / "数据显示..." / "我之前也踩了这个坑..." |
| **4-10/** | **核心干货** | 清单式拆解，每条独立金句 | "第1步..." / "误区3..." / 每条开头用数字或问题，结尾小总结 |
| **11-12/** | **高潮转折** | 个人实验 / 失败故事 / 惊喜 | "本来以为完了，结果..." / "这一步直接让我..." |
| **End** | **总结+CTA** | 升华价值 + 强行动呼吁 | "总结一下..." + "你最近翻车过吗？评论区说说" / "关注不迷路" |

**爆款技巧**:
- **视觉**: 多换行，短句为主，配合 Emoji。
- **互动**: 中间穿插提问 "你同意吗？"。
- **避坑**: 避免负面算法词，引导正面讨论。

## C. 长文 (Long Article)
*深度信任、品牌建设*

**结构**:
1.  **标题**: 好奇 + 利益 + 数字/年份 (e.g. "2026年后悔没早学的...")
2.  **开头 (200-400字)**: 极端故事 / 反常识宣言 / 翻车经历。
3.  **中间**: 小标题框架拆解 + 截图/案例。
4.  **结尾**: 哲学升华 + 自然植入(CTA)。
